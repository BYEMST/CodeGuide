- 首先，AgentNode 节点是由 ChatModelNode 节点流转过来的，每个节点的流转都是类似的操作，处理完业务功能后，则路由到下一个节点继续完成其他业务。
- 之后，AgentNode 是一个多 LlmAgent 装配的过程，我们在配置智能体的时候，有一些复杂的场景则要多个 LlmAgent 分别做不同的事情，来完成一个整体的流程。所以这类的智能体是多个 LlmAgent 配置。
