## 一、本章诉求

开发脚手架接口对接 UI，简单展示服务端能力。

在业务场景中，使用 AI Agent 脚手架，构建项目后所提供的服务接口，一般是附带着业务应用场景能力的。这些提供的能力接口，可以对接到客服、巡检、数据、量化、风控等各类场景使用。
