## 一、本章诉求

把构建的智能体，填充到 InMemoryRunner 中，以及注册到 Spring 容器中。这样后续就可以使用 Spring 容器中的对象，进行 AI Agent 对话了。

>目前是把最后构建的 SequentialAgent 写入到 InMemoryRunner 中，后续会做成动态的，让配置任何的一个智能体都可以填充到 InMemoryRunner 中。
